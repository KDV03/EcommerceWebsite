# ProbitX E-Commerce Platform - Velzon Design

## Modern E-Commerce Dashboard with Velzon UI Framework

### Features
- ✅ Modern Velzon-inspired dashboard
- ✅ Product management
- ✅ Order tracking
- ✅ Supplier management  
- ✅ Real-time metrics
- ✅ Responsive design
- ✅ Clean architecture

### Quick Start
1. Install .NET 8 SDK
2. Run: `dotnet restore`
3. Run: `dotnet build`
4. Run: `dotnet run`
5. Open: https://localhost:5001

### Project Structure
```
├── Controllers/     # MVC Controllers
├── Models/          # Data models
├── Views/           # Razor views
├── Data/            # Database context
├── Services/        # Business logic
└── wwwroot/         # Static files
```

### Technologies
- ASP.NET Core 8.0 MVC
- Entity Framework Core
